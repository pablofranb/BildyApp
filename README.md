# BildyApp API

Backend REST desarrollado con Node.js, Express y MongoDB para la gestión de usuarios y empresas.

## Tecnologías usadas
- Node.js
- Express
- MongoDB + Mongoose
- JWT
- bcryptjs
- Zod

Estas dependencias aparecen  en `package.json`. 

## Requisitos previos
- Node.js instalado
- npm
- Una base de datos MongoDB (local o MongoDB Atlas)

## Instalación
1. Clona el repositorio:
git clone https://github.com/pablofranb/BildyApp.git
cd BildyApp
